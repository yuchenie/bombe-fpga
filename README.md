# FPGA Project Website

## Structure

- `index.html`: Main webpage with project documentation
- `content/`: Directory for markdown content files
- `images/`: Directory for images and diagrams

## Content Management

The website loads text content from markdown files in the `content/` directory:

- `intro.md`: Project introduction
- `design.md`: High level design
- `implementation.md`: Implementation details
- `results.md`: Testing and results
- `conclusion.md`: Conclusion

Edit these `.md` files to update the content. The website will automatically load and render the markdown as HTML.
